## 📁 Projets Académiques

### 🔒 Plateforme IoT sécurisée ECOTRACK

Déploiement d'une infrastructure sécurisée sur serveur dédié **Hetzner / Proxmox VE** :
segmentation réseau par zones avec pfSense, mise en place d'un WAF **Nginx/ModSecurity**,
reverse proxy **HAProxy**, VPN OpenVPN, monitoring avec **Prometheus/Grafana** et SIEM **Wazuh**.

🔗 [Voir le projet](https://github.com/SALIHSIDIBE/ECOTRACK.git)

---

### 📞 Téléphonie sur IP avec Elastix

Installation et configuration d'un serveur PBX **Elastix** en environnement virtuel,
avec création des extensions SIP et tests de communication interne entre postes.

🔗 [Voir le projet](https://github.com/SALIHSIDIBE/Voip-elastix.git)

---

### 🌐 Réseau d'entreprise sur Cisco Packet Tracer

Conception et simulation d'une infrastructure réseau d'entreprise : adressage IP,
configuration des équipements Cisco, segmentation VLAN et mise en place des protocoles DHCP et DNS.

🔗 [Voir le projet](https://github.com/SALIHSIDIBE/Reseau-entreprise.git)
